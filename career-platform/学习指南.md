# 职引未来 - 学习指南

> 本文档帮助团队成员理解项目代码，便于后续学习和修改

## 一、项目运行步骤

### 1. 安装 Python

如果电脑没有安装 Python：
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本
3. 安装时勾选 "Add Python to PATH"

### 2. 启动项目

**方法一：双击启动脚本**
- 双击 `启动.bat` 文件
- 会自动安装依赖并启动服务器

**方法二：命令行启动**
```bash
# 打开 CMD 或 PowerShell，进入项目目录
cd C:\Users\24383\Desktop\career-platform

# 安装依赖
pip install flask flask-cors

# 启动服务器
python app.py
```

### 3. 访问网站

浏览器打开：http://127.0.0.1:5000

## 三、代码学习路径

### 第一阶段：HTML 基础（1-2 天）

**学习目标**：理解 HTML 页面结构

1. 打开 `templates/login.html`
   - 学习表单结构：`<form>`, `<input>`, `<label>`
   - 理解 Bootstrap 的类名使用

2. 打开 `templates/index.html`
   - 学习导航栏结构
   - 学习轮播图结构

**练习**：修改登录页的标题和按钮文字

### 第二阶段：CSS 样式（1-2 天）

**学习目标**：理解并修改页面样式

1. 打开 `static/css/style.css`
   - 学习 CSS 变量定义
   - 学习卡片、按钮等组件样式

**练习**：
- 修改主色调（把蓝色改成紫色）
- 修改字体大小

### 第三阶段：JavaScript 交互（2-3 天）

**学习目标**：理解前端逻辑

1. 打开 `static/js/main.js`
   - 学习 `fetch`  API 调用
   - 学习 localStorage 使用
   - 学习 DOM 操作

2. 打开 `templates/login.html` 底部的 `<script>`
   - 学习表单提交逻辑
   - 学习登录成功后的跳转

**练习**：修改登录成功后的提示信息

### 第四阶段：Python 后端（3-4 天）

**学习目标**：理解后端逻辑

1. 打开 `app.py`
   - 学习 Flask 路由定义
   - 学习 SQLite 数据库操作
   - 学习 API 接口设计

**重点函数**：
- `init_db()` - 数据库初始化
- `api_login()` - 登录接口
- `get_internships()` - 获取岗位列表
- `save_resume()` - 保存简历

**练习**：
- 修改测试账号的密码
- 添加一个新的实习岗位数据

### 第五阶段：数据库（1-2 天）

**学习目标**：理解数据库设计

查看 `app.py` 中的 `init_db()` 函数，了解各表结构：
- `users` - 用户表
- `internships` - 岗位表
- `resumes` - 简历表
- `applications` - 投递记录表
- `articles` - 文章表
- `assessments` - 测评结果表

**练习**：在数据库中手动添加一条用户记录

## 四、常见问题解答

### Q1: 端口被占用怎么办？

修改 `app.py` 最后一行：
```python
app.run(debug=True, port=5001)  # 改成 5001 或其他端口
```

### Q2: 如何重置数据库？

删除 `db/career.db` 文件，重启服务器会自动重建

### Q3: 如何修改首页的轮播图文字？

编辑 `templates/index.html`，找到这三段：
```html
<h2>找到你的理想实习</h2>
<h2>职业规划测评</h2>
<h2>学长学姐经验分享</h2>
```

### Q4: 如何添加新的实习岗位？

在 `app.py` 的 `insert_sample_data()` 函数中添加：
```python
(1, '新岗位标题', '岗位描述', '任职要求', '薪资', '城市', '实习', '2026-12-31'),
```
